Projeto acadêmico desenvolvido ao longo de um semestre na disciplina de Projetos de Sistemas Computacionais I (Sistemas de Informação - UFGD). 

Meu parceiro e eu criamos um jogo educativo de digitação voltado para todas as faixas etárias, pensado especialmente para a Smart Tech, escola de informática onde o projeto seria aplicado.
O sistema foi desenvolvido em Python, utilizando Tkinter para a interface, SQLite para registrar o desempenho dos usuários e JSON para organizar as frases usadas nos níveis. A interface é simples, intuitiva e acessível, projetada para facilitar a navegação e manter o foco na prática.
O jogo apresenta níveis com frases variadas, oferecendo feedback ao final sobre erros e acertos e permitindo o acompanhamento da evolução do usuário. A solução busca tornar o aprendizado da digitação mais envolvente e funcional.

O projeto foi implementado no site da Smart Tech e está em fase de testes e aprimoramentos contínuos. Ele integra conceitos de interface, lógica, manipulação de dados e experiência do usuário, resultando em uma aplicação prática e de fácil utilização.
